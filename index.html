About Ligma Browser

Ligma Browser is a fast, secure, and customizable web browser designed to enhance your browsing experience.

Key Features

  • Fast and efficient browsing
  • Customizable interface
  • Privacy protection
  • Support for extensions
  • You must have Python and install PyQt5 and PyQtWebEngine to use the desktop version.
    In CMD: pip install PyQt5 and pip install PyQtWebEngine
